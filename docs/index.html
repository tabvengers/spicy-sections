Panelset Element

The Panelset Element is an experiment in which “good ol’ well supported HTML” is conditionally presented with different affordances; either as a tab set, or with independent collapses (“disclosure widgets”) or exclusive collapses (“accordions”).

Installation

Import the OUIPanelset class and use it to define <oui-panelset>. 

<script type="module">
import { OUIPanelset } from 'https://raw.githack.com/tabvengers/spicy-sections/oui/src/element.js'

customElements.define('oui-panelset', OUIPanelset)
</script>

These instructions will be simplified once the package is published.

Use the <out-panelset> element to wrap flat sections of content. 

<oui-panelset>
	<h2>Mail</h2>
	<p>… Mail content …</p>
	<p>… Mail content …</p>

	<h2>News</h2>
	<p>… News content …</p>
	<p>… News content …</p>
	<p>… News content …</p>
</oui-panelset>
