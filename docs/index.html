Panelset Element

The Panelset Element is an experiment in which “good ol’ well supported HTML” is conditionally presented with different affordances; either as a tab set, or with independent collapses (“disclosure widgets”) or exclusive collapses (“accordions”).

Installation

Import the OUIPanelsetElement class and use it to define the <oui-panelset> element. Use <out-panelset> to wrap flat sections of content. 

<script type="module">

import { OUIPanelsetElement } from 'https://raw.githack.com/tabvengers/spicy-sections/oui/src/element.js'

customElements.define('oui-panelset', OUIPanelsetElement)

</script>

<oui-panelset>
	<h2>Mail</h2>
	<p>… Mail content …</p>
	<p>… Mail content …</p>

	<h2>News</h2>
	<p>… News content …</p>
	<p>… News content …</p>
	<p>… News content …</p>
</oui-panelset>
